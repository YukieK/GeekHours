## Overview

**Geek Hours** is a simple study time management tool.

## Prerequisites

* Ubuntu **16.04** or later

* Python **3.5** or later

* SQLite **3.11.0** or later

## Installing

Install from PyPI via pip.

```
pip3 install geekhours
```
